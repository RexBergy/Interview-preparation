# Interview-preparation

This is an interview preparation app.

To start run main.py in a terminal with an OPENAI_API_KEY set in environment variables.

Open the app on the localhost.

Give the app a desired role with the main objective you want work on. 
Tell it when do you want to start your plan and when is your end date (interview, etc.)
Tell it how much time you are willing to spend per day. 

The app will fill your calendar with tasks to do. The app will give tailored quizz.
